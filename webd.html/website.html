Lists in HTML:

The three types of lists in HTML are ordered, unordered, and description lists

ORDERED LIST
The HTML ol tag defines an ordered list. An ordered list can be numerical or alphabetical.

UNORDERED LIST
An unordered list starts with the ul tag. Each list item starts with the li tag.The list items will be marked with bullets (small black circles) by default.The CSS list-style-type property is used to define the style of the list item marker.
